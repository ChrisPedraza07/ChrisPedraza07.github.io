A Mystery in Sugarland

An action/mystery game set in a vibrant, sugary-treat–themed world that’s slowly slipping into chaos. You play as Officer Pinkglaze, a seasoned donut cop who thought he’d seen it all… until now.

This is the current solo game project I am developing, it is still very much in the early stages.

Battle Arena

Battle Arena is my first 3D game and as well as my first local co-op game. Two players on opposing sides of the arena battle to reach the other player's base to score points. Each player has Defending bots that will attack the other player as they try to reach the base.

I made the player and bot models in Blender and added simple animations to them. I created the arena in Unity and used C# to program the game mechanics. this project was a good introduction to 3D game development, Unity's new input sytem and the workflow between creating models in Blender and importing them into Unity.

Killer Candy

I wanted to make a 3rd person mob shooter somewhat inspired by autoshooters like Vampire Survivors; I love that style of gameplay but wanted to make something that requires a little more player involvment. My idea was to add 3rd person shooting into the mix as well as the addition of player triggered powerups along with autoshooting powerups and passive powerups. I really have no story for the candy theme, I just liked the idea of zombie candy, Adventure time / Wreck it Ralph inspired. Assets were created in blender and game engine is Unity.

Hacky Sack Attack

A mobile game where the player uses their thumbs to control each shoe in order to juggle the hacky sack and performs tricks. This is a prototype that currently has working gameplay, score counter, and some tricks. I'm not sure if I will complete this game, but I thought it was a fun idea for a mobile game so I went ahead and made this prototype.

Donut Cop

A 2D platformer where you play as a cop turned donut who's hunting down his disgruntled ex-partner for betraying him. All assets were hand drawn and animated in Unity. This was my first solo project/game that I made in Unity.

I learned a lot about game development through this project; concept creation, planning, asset creation, and ultimately how to use Unity. I also learned how to use C# to program the game mechanics.

Factory Escape

This 2D platformer is a mobile game that utilizes AR targets to load different levels. This was my first group project, my responsibilites were character creation/animation and I did some of the movement scripting.

I learned a lot about how to effectively work in a group and that communication is key to success.I also learned how AR can be implemented in different ways.
